Summary Equation

6CO2 + 6H2O + Light → C6H12O6 + 6O2
Light Reactions (Thylakoid): Photons excite chlorophyll, split H2O, produce ATP + NADPH + O2.
Calvin Cycle (Stroma): CO2 fixed by RuBisCO into G3P, using ATP + NADPH. 6 turns = 1 glucose.
